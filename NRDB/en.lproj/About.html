Quick Start
Ideally, there should not be any need to explain how this app works. If there is, let me know! :)

When you initially install NRDB, you will need to at least download the card data from netrunnerdb.com. You can optionally also download all available images in advance, useful when you plan on using the app while offline or on a slow GSM/Edge connection. Missing images will be automatically downloaded when needed.
