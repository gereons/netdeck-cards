Quick Start

Quand vous installez Net Deck pour la première fois, il va falloir télécharger les données des cartes. Vous pouvez aussi télécharger tout les images disponibles, ce qui permet d'utiliser l'application quand il n'y a pas de réseau, ou une connection Edge/3G passable. Les images manquantes seront automatiquement téléchargées quand il y en aura besoin.

Pour importer des decks d'autres sources, il existe trois moyens:

  • Connectez l'application à votre compte Dropbox, copiez les fichiers .o8d vers le fichier “Apps/Net Deck”, puis utilisez “Importer des Decks”.
  • Copier une deck list depuis Safari, Mail etc dans le presse papiers. Lors du prochain démarrage de l'application, il vous sera demandé si vous voulez importer ce deck.
  • Copiez depuis netrunnerdb.com o netrunner.meteor.com l'URL d'une decklist dans le presse papiers. Lors du prochain démarrage de l'application, il vous sera demandé si vous voulez télécharger et importer ce deck.
